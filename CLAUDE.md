# Project Documentation

## Overview
This project is designed to test how local LLMs manage various tasks and workflows. It includes documentation, configuration files, and example scripts to facilitate testing and experimentation.

## Structure
- `CLAUDE.md`: This file provides an overview of the project and its structure.
- `.aiignore`: Specifies files and directories to be ignored by AI tools.
- `src/`: Contains source code and scripts for testing.
- `README.md`: Main README file for the project.

## Usage
1. Clone the repository.
2. Navigate to the project directory.
3. Run the provided scripts to test different functionalities.
4. Refer to the documentation for detailed instructions on each component.
